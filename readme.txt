Being that there wasn't a plugin to ensure a good video embed of Rumble videos, Shortcodes for Rumble was born. Simple to use and developer friendly. Tested with WP Bakery, Elementor, and Divi page builders.

== Description ==

Simply upload this plugin like any other, and activate it. Go to any post/page to edit.
example shortcode:
`[rumblevid url="https://rumble.com/embed/videoid/?pub=1" poster=false overlay="/wp-content/uploads/video-placeholder.png" height="500px" width="100%"]`

== Installation ==

* Upload 'rumble-shortcode.zip' like any other plugin or upload the zip file contents to the '/wp-content/plugins/' directory
* Activate the plugin through the 'Plugins' menu in WordPress
* Edit Post/Page
* Insert shortcode

== Frequently Asked Questions ==

= Why do I need this? =

Other plugins do work and so does the Rumble embed code. However, we ran into an issues during client builds with some page builders that would not accept the embed code or rumble video page.

= Does this work on Multisite? =

Yes, It should work fine on multisite installations.
